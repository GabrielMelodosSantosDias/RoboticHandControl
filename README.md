# Controlling a Robotic Arm with Hand Gestures

Extension project developed within the Computer Engineering program at Universidade Santa Cecília (UNISANTA).

My name is Gabriel Melo dos Santos Dias, I’m a 1st semester student, and this repository showcases my participation and contributions to the project’s development.

The project uses computer vision in Python to detect hand movements and send commands to an Arduino, which controls a robotic arm.

Communication between Python and Arduino is done using the PyFirmata (StandardFirmata) library.
